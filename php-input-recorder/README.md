# PHP Input Recorder (Patient Zero)

This is the first project in the ** PHP Projects** series.

## Features
- Collects user input
- Displays input dynamically
- Clean UI with Flexbox
- Beginner-friendly PHP logic

## Requirements
- PHP 7+
- Local server (XAMPP, WAMP, MAMP)

## How to Run
1. Clone the repo
2. Move folder into `htdocs`
3. Start Apache
4. Visit:
   http://localhost/php-input-recorder/

## Concepts Learned
- PHP GET requests
- HTML forms
- PHP + HTML integration
- Basic CSS styling
